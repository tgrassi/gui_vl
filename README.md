- Clone    
`clone git@github.com:tgrassi/gui_vl.git`     
`cd gui_vl`
  
- Create environment     
`python -m venv testgui`    
`source testgui/bin/activate`     

- Install reqs     
`pip install -r pyLabSpec-0.3.2/pyLabSpec/GUIs/requirements.txt`     

- Launch and pray    
`python pyLabSpec-0.3.2/pyLabSpec/GUIs/qtfit.py`
